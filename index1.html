HTML

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat dan merancang halaman web. HTML menyediakan struktur dasar dari sebuah halaman web dengan menggunakan elemen-elemen seperti paragraf, judul, tautan, gambar, dan form. Setiap elemen ditulis dengan tag yang dikelilingi oleh tanda kurung sudut

Fungsi HTML

HTML berfungsi untuk menyusun dan menampilkan konten di halaman web, seperti teks, gambar, dan video, dengan memberikan struktur dasar menggunakan elemen dan tag. Ini memungkinkan pembuatan hyperlink untuk menghubungkan halaman web, mendukung formulir untuk mengumpulkan input pengguna, serta menggunakan tag semantik untuk mempermudah pemahaman struktur halaman oleh mesin pencari dan alat bantu aksesibilitas.

CSS

CSS (Cascading Style Sheets) adalah bahasa pemrograman yang digunakan untuk mendesain dan mengatur tata letak halaman web. CSS memungkinkan Anda untuk menentukan bagaimana elemen HTML ditampilkan, seperti warna, font, margin, dan posisi. Dengan CSS, Anda dapat membuat halaman web lebih menarik dan konsisten secara visual dengan mengaplikasikan gaya secara terpisah dari struktur HTML. Misalnya, Anda dapat mengubah warna latar belakang, jenis huruf, dan jarak antara elemen tanpa mengubah HTML itu sendiri.

Fungsi CSS

CSS (Cascading Style Sheets) berfungsi untuk mengatur desain dan tata letak halaman web, termasuk warna, font, dan jarak elemen, sehingga tampilan visual halaman dapat disesuaikan dengan mudah. Dengan CSS, Anda dapat memisahkan konten HTML dari gaya, menciptakan desain yang konsisten di seluruh halaman, serta membuat desain responsif yang menyesuaikan dengan berbagai ukuran layar. CSS juga mendukung animasi dan transisi untuk efek visual yang lebih dinamis.
